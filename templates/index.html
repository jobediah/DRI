Welcome to my tutorial on API and data visualization using College Scorecard API and D3.js


It seems that big data and data science are technology's buzzwords of the year. New technologies and improved global connectivity are creating large, unstructured data sets that offer a wealth of information. However, the management of big data brings with it several issues, including the acquistion, storage, searching, analysis, and visualization these large datasets. Cloud platform solutions seem to go hand in hand with big data's challenges. It offers storage and computing power for analysis of datasets all on the same platform to provide low latency for data visualization. The end product, inferences incurred from viewing the data as a chart or graph, provides a huge impact for individuals and companies.


Several companies have produced highly profitable products based off data analysis and visualization. Where does that leave the individual? Fortunately, the wealth of data is so abundant that many agencies are offering public access to a myriad of data, which the individual can freely obtained through an application program interface (API). Large datasets are free for someone to explore and create their own app.


The goal of this tutorial is to provide a guide on how to create your own cloud-based application using Google's App Engine cloud platform, College Scorecard API datasets, and D3.js for data visualizaiton. You'll request a dataset, parse it, filter it, and display it as a chart all on the cloud!


Start at the getting started page to learn how to create your own cloud-based data analysis application. Or, take a look at the demo page to view the end product.
